# Screeps中文笔记

## 中文文档

[中文文档](https://screeps-cn.github.io)，由国内玩家自发维护，包含 api 文档。已完全汉化。

在玩Screeps之前，尽量看一遍官方文档。然后通过官方的[新手教程](https://screeps.com/a/#!/sim/tutorial)。如果全部通过，说明你很适合这个游戏，那么就开始搭建环境吧。
